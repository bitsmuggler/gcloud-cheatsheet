# Gcloud Cheatsheet

## Authentication

| Command  |  Description |
|---|---|
| `gcloud auth login`  | Login into gcp (+ add new login account) |
| `gcloud auth list` | List all authenticated accounts


## Appengine

| Command  |  Description |
|---|---|
| `gcloud init`  | Initialize gcp sdk |
| `gcloud app deploy` | Deploy the app to the gcp
